# 🤖 AI Recommendation App

Fullstack AI-powered recommendation system.

## 🚀 Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: SQLite
- AI: Groq (LLaMA 3)

## ✨ Features
- Mood-based recommendations
- Multi-language support (TR / EN)
- Countdown timer
- History tracking

## 🛠️ Setup
```bash
npm install
npm start
